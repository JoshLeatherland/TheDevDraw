## ğŸš€ Running Locally

This repository is a workspace containing three projects:

- tdd-components
- tdd-tools
- tdd-app

tdd-app is the main application.

### Install dependencies

```bash

```

### Build workspace packages

```bash
npm run build
```

### Run the app

```bash
cd packages/tdd-app
npm run dev
```
